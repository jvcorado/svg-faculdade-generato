## Como rodar?

Clone ou faça o download deste repositório. Acesse a pasta, instale as dependências e execute o via nodemon o javascript `server.js`  

```bash
git clone https://github.com/mmamorim/svguid.git
cd svguid
npm install
npm run dev
```
